## 📌 Sobre o Projeto

Este projeto é um **aplicativo Gerador de Frases Aleatórias** desenvolvido para fins educacionais usando Android Studio com Java e Kotlin. O objetivo é demonstrar habilidades fundamentais de desenvolvimento Android, incluindo design de interface e lógica de randomização, com uma experiência simples e intuitiva.

> 🎯 Foco em UI, randomização, botões interativos e estrutura Android.

---

## ✨ Funcionalidades

| Recurso | Descrição |
|---|---|
| 📝 **Título Centralizado** | "Gerador de Frases" exibido na tela principal |
| 🎲 **Botão de Random** | Gera uma frase pré-definida aleatória ao ser clicado |
| 📋 **Listar Todas** | Botão para exibir todas as frases disponíveis no repositório |
| 🎨 **Interface Simples** | UI intuitiva para uso imediato |
| 📱 **Nativo Android** | Desenvolvido com Java e Kotlin |

## 🚀 Como Executar

### Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- ✅ [Android Studio](https://developer.android.com/studio) (versão recente)
- ✅ SDK do Android configurado
- ✅ Emulador ou dispositivo físico Android

### Passo a Passo

```bash
# 1. Clone o repositório
git clone https://github.com/RDEsley/Gerador-Frases-Aleatorias.git

# 2. Acesse a pasta do projeto
cd Gerador-Frases-Aleatorias
```

```
3. Abra o Android Studio
   → File > Open > selecione a pasta do projeto clonado

4. Aguarde o Gradle Sync finalizar
   → Todas as dependências serão baixadas automaticamente

5. Selecione seu dispositivo/emulador e clique em ▶ Run
```
